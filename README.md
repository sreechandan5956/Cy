## Quick Start

### Prerequisites
- Node.js (v14 or higher)
- PostgreSQL database
- Email account for OTP (Gmail recommended)

### Installation & Setup

1. **Clone the repository**
```bash
git clone <repository-url>
cd <project-directory>
```

2. **Install dependencies**
```bash
cd backend
npm install
```

3. **Configure environment**
```bash
cp backend/.env.example backend/.env
# Edit backend/.env with your database and email credentials
```

4. **Setup database**
```bash
bash setup_postgres.sh
```

5. **Start the application**

On Linux/Mac:
```bash
bash start.sh
```

On Windows:
```bash
start.bat
```

Or manually:
```bash
cd backend
npm start
```

6. **Access the application**
Open your browser and navigate to: `http://localhost:5000`

### Features
- User registration with email verification (OTP)
- Secure authentication with JWT tokens
- Protected dashboard and routes
- Cybersecurity learning challenges
- Multiple security domains (Web Security, Cryptography, Forensics, etc.)
